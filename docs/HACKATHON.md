# Hackathon Summary

MedBoard was built in 48 hours during MedFusion HackFest.

## Summary

Built an AI-powered disease surveillance dashboard that uses global health data and analytical components to monitor disease risks and visualize outbreak trends.

The project involved work across:

- API integration
- Backend systems
- Frontend dashboard development
- Health data visualization
- Disease risk and outbreak monitoring

The team qualified for the final round.

## Event

MedFusion HackFest, organized by ISCB-SC RSG-India, Telangana Node at Mahindra University.
